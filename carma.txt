Criei um novo repositorio mas não consigo fazer o clone. Dá-me o erro
"fatal:Couldn't find remote ref HEAD"

RESOLVIDO ->É um erro que ocorre em repositórios vazios.
Tem de ser corrido o código:
Next steps:

  mkdir rep
  cd rep
  git init
  touch README
  git add README
  git commit -m 'first commit'
  git remote add origin git@github.com:user/repositorio.git
  git push -u origin master
